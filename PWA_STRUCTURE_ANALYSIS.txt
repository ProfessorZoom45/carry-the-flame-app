Important design decision
-------------------------
The PWA should NOT keep card art as one giant flat image folder forever.
The card database already points toward a cleaner structure like:
images/ANM/reese_the_great_s_gundam.gif
images/DBZ/goku.gif
images/BL1/yoichi_isagi.gif

So the app should use:
images/<SET>/<card_image_file>

Why this matters
----------------
1. Faster lookup.
2. Cleaner GitHub repo.
3. Easier debugging.
4. Easier missing-image audits.
5. Cleaner future CDN or app wrapping.
6. Better long-term PWA caching.

Mechanics source of truth
-------------------------
Use CTF Beginners Guide v7 as the source for gameplay requirements.
Key locked requirements for the PWA:
- CTF supports 2 to 4 players.
- Main win paths are Chi KO, 7 Kills, and 7 Extractions.
- Main Deck size is 40 to 60 cards.
- Fusion Deck size is 0 to 15 cards.
- Starting hand is 5 cards.
- Turn order uses 7 locked phases.
- Starting Chi should be treated consistently as 10,000 for current CTF app work per user rule, even if older guide text says 9,999.
- Great Cards cannot enter the Box.
- Libra Zone cards are treated as Trick cards and cannot be used as End Phase sacrifices.
- Palm Tricks cannot be used as counter-response cards.
- Concealed Tricks are the reactive opponent-turn cards.
